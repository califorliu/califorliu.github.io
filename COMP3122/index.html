AI Learning Assistant

Progressive Disclosure Search & Generation RAG System

Project Overview

This project focuses on an AI assistant for learning support. The system helps students learn more effectively by generating personalized step-by-step guidance based on both course materials and the student's current knowledge level.

Proposed Workflow

  1. Build a course knowledge base: This may include teaching materials, lesson plans, and past course content.
  2. Understand students' knowledge levels: This could be based on their assignments, prompt history, common questions, and a short pre-class survey.
  3. Generate step-by-step learning guidance: Based on existing frameworks such as online tutorials and teacher-designed structures, the AI creates customized learning paths for students.

MVP Goals

1. Index Building

Enable file uploads, knowledge point extraction, and formatted storage.

2. Search Optimization

Implement different search depths for different levels of detail (routes) using heading levels for chunking, supporting mixed searches (vectors and keywords).

3. Generation

Reorder the priority of retrieved chunks, remove duplicates, generate output for different routes, and ensure that further questions can match detailed sub-chunks.

4. Module Integration

Comprehensive UI and UX design to create a seamless user experience.

Key Features

  • Lesson plan uploads and knowledge base construction
  • Knowledge framework generation
  • Advanced question-and-answer functionality
  • Course separation and organization
  • Mock exam generation
